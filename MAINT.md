LazyDictionary
==============

Install
-------

    git clone git@github.com:janrain/lazydict.git
    cd lazydict
    sudo ./setup.py install


Test
----

    ./setup.py test


Distribute
----------

	pip install twine
    ./setup.py sdist
    twine upload dist/*


Versioning
----------
This software follows the Semantic Versioning convention.
http://semver.org/
